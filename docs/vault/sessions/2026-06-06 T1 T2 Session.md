## T1 — Bugs Fixed This Session

### Three bugs causing 0% success (F-11)

All three discovered by reading the official `gr00t/eval/sim/LIBERO/libero_env.py`:

**1. Rotation convention: Euler → Axis-Angle**

The model was trained with axis-angle. We were passing Euler angles from scipy. The orientation input was in the wrong space, producing garbage actions.

```python
# Before (wrong)
euler = Rotation.from_quat(xyzw).as_euler("xyz")

# After (correct)  
den = np.sqrt(1.0 - quat[3] ** 2)
rpy = (quat[:3] * 2.0 * math.acos(quat[3])) / den
```

**2. Image not flipped**

LIBERO renders images in a flipped coordinate system vs GR00T training. Both spatial axes must be flipped.

```python
# Before (wrong)
img = obs_dict["agentview_image"]

# After (correct)
img = obs_dict["agentview_image"][::-1, ::-1]
```

**3. Gripper action transform**

GR00T outputs `[0,1]` (LeRobot: 0=close). LIBERO needs `[-1,+1]` inverted.

```python
# Before (wrong)
action[-1] = raw_gripper_value

# After (correct)
action[-1] = -np.sign(2.0 * action[-1] - 1.0)
```

**Impact:** Fixing all three took success from 0% → 97%.

### Other bugs fixed earlier

| Bug | Fix |
|---|---|
| `Gr00tPolicy.eval()` → `AttributeError` | Call `policy.model.eval()` instead |
| Wrong language key `"human.action.task_description"` | Correct: `"annotation.human.action.task_description"` |
| State as flat 8-dim tensor → `KeyError: Joint group 'x' not found` | Pass per-key dict: `{"x":(1,1,1), ..., "gripper":(1,1,2)}` |

## T2 — World Model for Manipulation

### What Was Built

New module: `programs/t2_manip_wm/`

- `collect_manip_rollouts.py` — runs GR00T on LIBERO, saves `(obs_state, action, reward)` per step as `.pt` dataset
- `train_wm_manip.py` — trains Dreamer-mini RSSM on that dataset, writes result MD
- `run_t2.sh` — one-shot pipeline: collect → train → commit → push

### Pipeline

```
GR00T policy (pre-trained) → LIBERO env (10 tasks × 20 eps)
    ↓
200 episodes saved as manip_rollouts_groot.pt
obs: eef_xyz(3) + axis_angle(3) + gripper(2) = 8-dim
act: 7-dim OSC delta
rew: 0.0 per step, 1.0 at terminal step if success
    ↓
Dreamer-mini RSSM (deter=128, stoch=32, hidden=128)
trained for 3000 steps on GPU
    ↓
Checkpoint: wm_manip.pt
Results: docs/results/t2_manip_wm.md
```

## Key Lesson

**Always read the official sim wrapper before writing your own obs adapter.**
GR00T's `libero_env.py` contained all three critical conventions (axis-angle, image flip, gripper transform) that took ~6 hours of debugging to discover through 0% success runs.
